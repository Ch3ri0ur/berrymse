Media Source Extensions Live Demo

Copyright 2020 Chris Hiszpanski

This is a demo of low-latency video streaming using the Media Source Extensions API. The API is supported by all major browsers, with the notable exception of Safari on most iOS devices. The API is supported by Safari on iPad devices with iOS 13 or newer.

The server receives a H.264 Network Abstraction Layer (NAL) unit byte-stream and wraps the NAL units within MPEG-4 ISO BMFF (ISO/IEC 14496-12) compliant boxes.
